# Residential-Energy-Consumption-Forecasting
This repository contains the dataset and the scripts used in the paper “Residential Energy Consumption Forecasting in Electric Utilities: An Approach Based on Random Forests and Time Series”, submitted to IEEE Latin America Transactions.

Manuscript IEEE LATAM ID: 10246

Instrucciones para ejecutar en google colab

-	Cargar los cuadernos .ipynb de la carpeta Programas a google colab
-	Cargar datos desde la carpeta Datos (CON_TEMPU1.csv)
-	Ejecutar Prediccion_por_Medias_Moviles_(unrc).ipynb para obtener el resultado dado en consumo_real_vs_pred_mm_usuario_10_2024.csv
-	Ejecutar Prediccion_con_ARIMA_(unrc).ipynb para obtener el resultado dado en consumo_real_vs_pred_arima_usuario_10_2024.csv
-	Ejecutar Prediccion_por_RF_(unrc).ipynb para obtener el resultado dado en consumo_real_vs_pred_mm_usuario_10_2024.csv
